# Techium - CSS Advanced Project

## Description

Techium is a complete website design project built as part of the Holberton School CSS Advanced curriculum. This project demonstrates progressive CSS exercise implementation across 32 cumulative CSS files, building a fully styled and responsive website.

## Features

- **Responsive Grid System**: Float-based layout with col-1-3 and col-1-2 classes
- **CSS Variables**: All styling driven by custom properties in :root
- **Dark Theme**: Dynamic styling with data-section-theme attribute
- **Smooth Animations**: Transitions and transforms with cubic-bezier timing
- **Normalize.css**: Browser consistency v8.0.1
- **Google Fonts**: Open Sans and Raleway typography
- **Component Styling**: Services cards, work overlays, testimonials, buttons

## Installation

1. Clone or download the project
2. Navigate to the CSS_advanced directory
3. Add images to the `images/` folder
4. Open `index.html` in a web browser

## CSS Exercises Overview

This project implements 32 progressive CSS exercises:

- **Exercises 1-8**: Foundation (scroll behavior, colors, variables, fonts)
- **Exercises 9-13**: Typography and pseudo-classes
- **Exercises 14-21**: Layout and grid system
- **Exercises 22-29**: Components (dark theme, cards, navigation)
- **Exercises 30-32**: Advanced effects (overlays, decorations, animations)
